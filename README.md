# mern-thinkingboard

Full-stack MERN note-taking app with an Express/MongoDB backend and a React/Vite frontend.

## Structure

- `backend`: REST API for notes, database config, and rate limiting.
- `frontend`: React client for listing, creating, editing, and deleting notes.

## Scripts

Backend:

```bash
cd backend
npm run dev
```

Frontend:

```bash
cd frontend
npm run dev
```
